# KYC Verification & Risk Analysis System

## 📌 Project Overview

The **KYC Verification & Risk Analysis System** is designed to identify and classify customer risk levels based on KYC (Know Your Customer) parameters. This system simulates real-world **AML/KYC processes** used in financial institutions and consulting firms like Deloitte.

The system analyzes customer data and classifies them into:

- 🟢 Low Risk
- 🟡 Medium Risk
- 🔴 High Risk

This project helps identify suspicious customer behavior and potential fraud risk.

---

## 🎯 Features

- Customer Data Analysis
- Risk Level Classification
- Suspicious Transaction Detection
- KYC Verification Logic
- Risk Dashboard Visualization
- CSV Report Generation
- Easy to Extend and Modify
